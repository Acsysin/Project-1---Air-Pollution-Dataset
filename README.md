The dataset used for this project is the Air Pollution Dataset from UC Irvine's Machine Learning Repository
https://archive.ics.uci.edu/dataset/360/air+quality


Three models are run on this dataset and a final output to analyze data accuracy has been given. 
However, a data leakage problem ended up leading to innacurate results, so the data leakage problem has been fixed, and retraining and reevaluation has been performed at the very end.
The original and final output post correction have been attached as screenshots alongside the code file.
